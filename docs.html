Frontend & Backend Routes

Frontend Routes:
GET  /               → landing.html
GET  /login          → login.html
GET  /signup         → signup.html
GET  /components     → components.html

Backend API Routes:
POST /api/login      → { email, password } → { token, user }
POST /api/signup     → { fullname, email, password } → { user }
